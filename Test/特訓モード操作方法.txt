# 特訓モード
### 操作方法(停止時)

|     キー       |          動作           |
| :------------: | :---------------------: |
|     Space      |        譜面の再生       |
| 1P縁/PgUp/PgDn |        譜面の移動       |
|      2P縁      |      再生速度の変更     |
|       A        |  ジャンプポイントの追加 |

※再生時はSpaceで譜面の停止ができること以外普段通りのキー操作となります。  
　ジャンプポイントに移動する際はConfig.ini記載の時間内で``1P縁``を5回連打してください。
